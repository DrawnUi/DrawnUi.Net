MAUI Tutorials

These tutorials are the .NET MAUI tutorial lane for DrawnUI.

They assume a MAUI app host, DrawnUi.Maui, and a DrawnUI Canvas inside MAUI pages. They are still useful for understanding shared DrawnUI concepts, but they are not the primary onboarding path for Blazor or DrawnUi.Net.

If you are choosing a host first, start with Platforms and Packages.

Beginner tutorials

  • Your First DrawnUI App (XAML) - Complete setup from scratch, basic controls and layouts, simple animations, and MAUI deployment.
  • Your First DrawnUI App (C# Fluent) - The same app built with fluent C# syntax and DrawnUI observation patterns.

Intermediate tutorials

  • Interactive Cards - Animated cards, gesture handling, effects, and reusable composition patterns.
  • A Custom Drawn Control - A custom game-style button with composed visuals and press feedback.

Advanced tutorial

  • News Feed Scroller - Data binding, recycled cells, virtualization, and large-data-list patterns.
